rgo

Riak Go binding based around interoperability with both HTTP and protocol buffer support. The aim is to provide a relatively unified interface for both interfaces.

Installation

Recommended only for development of the library in the current state. Once completed installation will be by:

go get http://github.com/RadioactiveMouse/rgo

Current Status:

Partially complete with many things still broken/not implemented. Not suitable for use in development or production.
